# Todo List Web App - Backend

## Getting Started

First, install the dependencies:
```bash
yarn
```

Next, run the development server:

```bash
yarn start:dev
```

## Tech Stack
- TypeScript
- [Express](https://expressjs.com/)
- [Mongoose](https://mongoosejs.com/docs/)
